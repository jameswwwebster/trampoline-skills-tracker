# Cheatsheet Images

This directory should contain JPG images of each cheatsheet page in a `pages/` subdirectory.

## Required File Structure

Place your JPG files in: `resources/requirement-cheatsheets/pages/`

**Required file names (exact naming is important):**
- `page-1.jpg` - DMT National Pathway (page 1 of PDF)
- `page-2.jpg` - DMT Regional Pathway (page 2 of PDF)
- `page-3.jpg` - DMT Disability (page 3 of PDF)
- `page-4.jpg` - TRA Disability (page 4 of PDF)
- `page-5.jpg` - Club Level (page 5 of PDF)
- `page-6.jpg` - Club Level - North Region (page 6 of PDF)

## File Naming Convention

The files **must** be named exactly:
- `page-1.jpg`
- `page-2.jpg`
- `page-3.jpg`
- `page-4.jpg`
- `page-5.jpg`
- `page-6.jpg`

And placed in the `pages/` subdirectory within this folder.

## Image Requirements

- Format: JPG/JPEG
- Quality: 90% or higher recommended for readability
- Resolution: 300 DPI or higher recommended
- File size: Optimize for web (typically 500KB - 2MB per page)
